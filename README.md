# precheck — 开仓前多维体检工具

一个命令行 + Claude Code skill 工具:输入一个合约代码,输出多周期趋势、量价、
持仓量、资金费率、关键位、支撑阻力互换位、动能衰竭等客观数据,并给出参考交易方案。

## ⚠️ 免责声明

**本工具只提供客观数据快照和启发式参考,不是交易建议,不构成任何投资推荐。**

- 自动生成的交易方案(入场/止损/止盈/仓位)是基于固定规则的机械计算,**不要机械跟随**。
- 触达概率、期望值是启发式估算,**必须用你自己的复盘数据校准**。
- 趋势线/形态等复杂结构的识别,数值算法不可靠,**以你自己看图为准**。
- 交易有风险,盈亏自负。

## 安装

需要 Python 3.10+。

```bash
# 把 precheck/ 目录放到你的项目下, 在其父目录运行:
python3 -m precheck BTCUSDT

# 可选: 装 certifi 避免 macOS 的 SSL 证书问题 (没装也能跑, 有 fallback)
pip install certifi
```

需要能访问 `fapi.binance.com`(币安合约公开 API)。若所在地区被墙,需自备代理或在
可访问的服务器上运行。工具只读公开行情,**不需要 API key,不碰你的账户**。

## 用法

```bash
python3 -m precheck <SYMBOL> [-l 级别] [--account 资金] [--risk 风险%] [--okx] [--check]
```

**交易级别 `-l` / `--level`**(决定关键位用哪些周期、多远还算数;持仓时间随数字递增):

| 级别 | 名称 | 持仓 | 关键位周期 | 距离基准 |
|------|------|------|-----------|---------|
| `0` | 日内(默认)| 几小时~1天 | 1h + 4h | 4h ATR × 6 |
| `1` | 波段 | 几天~2周 | 4h + 日线 | 日线 ATR × 6 |
| `2` | 趋势 | 数周以上 | 日线 + 周线 | 日线 ATR × 10 |

也支持名字:`-l swing` / `-l 波段`。

**示例:**
```bash
python3 -m precheck BTCUSDT              # 日内(默认)
python3 -m precheck ETHUSDT -l 1         # 波段
python3 -m precheck SOLUSDT -l 2         # 趋势
python3 -m precheck BTCUSDT --okx        # 用 OKX 数据源(默认币安)
python3 -m precheck BTCUSDT --check      # 加交互式 5 项开仓 checklist
python3 -m precheck BTCUSDT --account 5000 --risk 1   # 自定义账户/风险
```

**输入格式**兼容:`BTCUSDT` / `BTC-USDT-SWAP` / `BTC/USDT` / `btcusdt` 等。

## 输出说明

- **[1] 多周期 EMA/ATR** — 15m/1h/4h 的趋势结构、均线偏离度
- **[2] 量价关系** — 当前(已收盘)K 线放量/缩量 + 涨跌方向
- **[3] 持仓量 OI** — 价 vs OI 的强弱含义
- **[4] 资金费率** — 多空拥挤度(极端值是风险预警)
- **[5] 大级别压力/支撑位** — 4h+日线摆动点 + 成交密集区
- **[5b] 支撑阻力互换位** — 前低聚集→上方压力、前高聚集→下方支撑,每带给保守/中庸/激进三档
- **[6] 近端动能衰竭** — 见顶/见底的放量反向、长影线、量价背离
- **综合评分** — Bias(趋势方向) + Timing(入场时机:可追/等回踩/观望/反转预警)
- **交易方案** — 系统按客观规则给的 LONG/SHORT 参考方案(含 R/R、估算期望值)

## 数据源

默认币安 USDⓈ-M 永续合约(`fapi.binance.com`),`--okx` 切 OKX。均为公开只读接口,
无需认证。

## 已知局限

- OI/资金费率是实时接口,无历史值(不影响方向判断,仅作风险提示)。
- 交易方案的胜率/期望是启发式估算,不是实盘统计,仅供参考。
- 不做趋势线/形态的自动识别(经验证不可靠),这些请自己看图。
