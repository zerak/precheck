# 开仓前多维体检 + 图表综合判断

运行 `python3 -m precheck $ARGUMENTS` 获取多维度数据诊断。
在**含有 `precheck/` 子目录的那个目录**下运行(通常就是当前工作目录), 直接运行即可, **不要 cd 到别处**。

可加 `-l` / `--level` 指定交易级别:0 日内(默认)/ 1 波段 / 2 趋势
—— 决定关键位用哪些周期、多远还算数。不带则默认日内(1h+4h)。

## 工作流程

1. 运行 precheck,获取多维数据(EMA/ATR、量价、OI、Funding、关键位、动能)+ 自动方案
2. 如果用户同时贴了 K 线图,结合图表做综合判断:
   - 图表上的 K 线形态(pin bar、engulfing、doji 等)
   - 关键水平位的假跌破/假突破回收
   - 成交量柱的视觉确认
   - 趋势线 / 通道 / 形态(三角形、旗形等)—— 以人眼看图为准,工具不自动识别
3. 给出最终综合建议:
   - precheck 数据说了什么(客观)
   - 图表形态说了什么(主观但重要)
   - 两者是否一致 → 信号强度评级
   - 如果冲突 → 指出矛盾点,建议等待

## 输出格式

### 数据面(precheck 输出摘要)
- 多周期方向 + 共振/分歧(Bias)+ 入场时机(Timing)
- 量价/OI/Funding 关键信号
- 关键位 / 支撑阻力互换三档
- 自动方案的 R/R 和估算期望值

### 图表面(如有图片)
- 当前 K 线形态识别
- 关键位行为(假突破?回收?放量确认?)
- 大结构判断(趋势/震荡/筑底/筑顶)

### 综合判断
- 方向倾向 + 信心等级(强/中/弱)
- 具体入场条件(什么情况下可以做)
- 不做的条件(什么情况下放弃)

## 重要提醒

- 自动方案是机械计算的**参考**,不是交易建议,不要机械跟随。
- 触达概率/期望值是启发式估算,需用自己的复盘数据校准。
- 趋势线/复杂形态请自己看图,工具不做自动识别。
